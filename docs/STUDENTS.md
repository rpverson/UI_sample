# Guía rápida (alumnos)

## Objetivo

Completar retos de UI usando el builder visual (tags + clases Tailwind) y validarlos con tests.

## Flujo recomendado

1. Instala dependencias: `npm install`
2. Levanta el entorno: `npm run dev`
3. Selecciona el reto en la UI.
4. Construye el componente desde el mapa: añade etiquetas, contenido y clases.
5. Comprueba la sección "Validacion del Reto".
6. Ejecuta `npm test` y repite hasta que todo pase.

## Consejos

- Usa etiquetas correctas: `button`, `a`, `input`, `header`, `nav`, etc.
- Agrega estados: `hover:*`, `focus-visible:*`, `disabled:*` cuando aplique.
- No “hackees” los tests: el objetivo es aprender criterios reales (UX/A11y y estructura semantica).
